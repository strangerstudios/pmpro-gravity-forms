Integrate Gravity Forms with Paid Memberships Pro

== Description ==

= General features of this Integration include: =
1. Add users to specified levels after submitting a form.
1. Restrict forms for specified levels.

== Installation ==

= Prerequisites =
1. You must have Paid Memberships Pro and Gravity Forms installed and activated on your site.

= Download, Install and Activate! =
1. Download the latest version of the plugin.
1. Unzip the downloaded file to your computer.
1. Upload the /pmpro-gravity-forms/ directory to the /wp-content/plugins/ directory of your site.
1. Activate the plugin through the 'Plugins' menu in WordPress.
